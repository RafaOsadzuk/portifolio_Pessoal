Procurando um desenvolvedor FullsStack?

Rafael Souza Osadzuk Transformando ideias em código e desafios em soluções.

Como um futuro engenheiro de software em formação, concentro meus esforços no desenvolvimento web e busco minha primeira experiência profissional para testar minhas habilidades e enfrentar desafios que impulsionem meu crescimento nesta área. Tenho experiência como desenvolvedor full-stack em React.js, PHP, Java e Node.js, e já trabalhei em projetos pessoais como uma página sobre Star Wars, utilizando uma API, e uma calculadora de IMC em PHP, baseada em dados de um site. Estou constantemente atualizado com as novas tecnologias do mercado, mantendo um foco dedicado ao aprendizado contínuo.

Esse projeto traz várias informações sobre os personagens da franquia Star Wars. Essa aplicação possui uma paginação onde é possível navegar em meio a mais de 80 personagens. Ao clicar em um personagem, é aberto um modal com maiores informações sobre ele. Todas essas informações estão sendo obtidas através da API Swapi. Foram colocados em prática consumo de API e manipulação de elementos através de Javascript.

Calculadora de Indice de massa corporal feito em HTML, CSS e PHP onde ela pega as informações do usuário como Data de nascimento, peso e altura e faz o calculo do seu percentil, também a idade do usuário é relevante para o diagnostico, foi feito o site com base nos dados fornecidos por aqui: https://www.calcule.net/saude/tabela-imc/

Meu primeiro site criado, com base no curso Profissão programador feito apenas com HTML e CSS. Ele é como se fosse um site para a venda de ingressos para assistir ao filme, mostrando trailer, atores, comentarios sobre o filme, fotos, e uma parte para Contatos
